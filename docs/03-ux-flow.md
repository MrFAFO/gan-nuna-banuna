# 03 - UX Flow

# תזרים UX - גן נונה בנונה

## פתיחת אפליקציה

1. המשתמש פותח את האפליקציה.
2. מוצג מסך פתיחה / התחברות ממותג של גן נונה בנונה.
3. אם המשתמש לא מחובר, הוא רואה אפשרויות כניסה.
4. אם המשתמש מחובר, האפליקציה מזהה את סוג המשתמש:
   - הורה
   - צוות גן
5. המשתמש מועבר למסך הבית המתאים.

## תזרים גננת

פתיחת אפליקציה
→ התחברות כצוות הגן
→ מסך בית גננת

במסך הבית הגננת רואה:

- סיכום נוכחות.
- התראות חוזים.
- דיווחי היעדרות מהורים.
- כפתורי פעולה מרכזיים.

### ניהול ילדים

מסך בית גננת
→ רשימת ילדים
→ פרטי ילד / הוספת ילד

פעולות:

- חיפוש ילד.
- צפייה ברשימת ילדים.
- הוספת ילד.
- צפייה בסטטוס נוכחות וחוזה.

### נוכחות

מסך בית גננת
→ נוכחות
→ סימון סטטוס לכל ילד
→ שמירה

סטטוסים:

- הגיע.
- לא הגיע.
- מאחר.
- יצא מוקדם.

### סיכום יום

מסך בית גננת
→ סיכום יום
→ בחירת ילד
→ מילוי אוכל / שינה / מצב רוח / הערה
→ שמירה

### חוזים

מסך בית גננת
→ חוזים
→ העלאת חוזה
→ בחירת ילד והורה
→ שליחה לחתימה
→ מעקב סטטוס חתימה

סטטוסים:

- טיוטה.
- נשלח לחתימה.
- נצפה.
- נחתם.
- נדחה.
- פג תוקף.

## תזרים הורה

פתיחת אפליקציה
→ התחברות כהורה
→ מסך בית הורה

במסך הבית ההורה רואה:

- שם הילד.
- סיכום יום אחרון.
- הודעה חשובה.
- דיווח נוכחות אחרון.
- חוזה שממתין לחתימה, אם קיים.

### צפייה בסיכום יום

מסך בית הורה
→ סיכום יום
→ צפייה בפרטי היום

ההורה רואה:

- אוכל.
- שינה.
- מצב רוח.
- הערת גננת.

### דיווח היעדרות

מסך בית הורה
→ דיווח היעדרות
→ בחירת סטטוס
→ שליחה

אפשרויות:

- הילד לא מגיע היום.
- הילד יגיע מאוחר.
- הילד יצא מוקדם.

### חתימה על חוזה

מסך בית הורה
→ התראת חוזה
→ פתיחת מסך חוזה
→ פתיחת קישור חתימה
→ חתימה אצל ספק חיצוני
→ עדכון סטטוס באפליקציה
→ מייל אישור להורה ולגננת

## עקרונות UX

- פעולה אחת מרכזית בכל מסך.
- כפתורים גדולים וברורים.
- מינימום הקלדה.
- עברית מלאה.
- RTL.
- שפה פשוטה ולא טכנית.
- מסך בית כמרכז עבודה.
